## Browser Native Validation

브라우저 기본 검증 메시지는 현재 앱 문구가 아니라 브라우저/OS 언어 설정에 따라 표시됩니다.

예시:

| Current Message | Suggested Direction |
| --- | --- |
| `이 입력란을 작성하세요.` | 브라우저 기본 검증을 유지하거나, 추후 커스텀 인라인 에러 UI로 교체 |
